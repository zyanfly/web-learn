# 前端技术栈学习知识库

## 📖 项目说明

这是一个**交互式前端学习知识库**，通过问答的方式帮助用户巩固前端知识点。

### 🎯 核心功能

1. **分类问答学习**：按技术栈分类（JS、CSS、HTML、React、Vue、TypeScript 等）进行问答练习
2. **错题追踪**：自动记录答错的问题，便于后续复习
3. **智能复习**：在后续练习中有计划地穿插之前答错的知识点
4. **跨平台同步**：通过 Git 同步，可在任何设备上继续学习

## 🤖 AI 模型使用指南

> **重要**：本节专门写给其他 AI 编程助手，确保你能无缝接手这个项目。

### 项目结构

```
learn/
├── README.md                    # 你正在读的文件
├── .agent/workflows/quiz.md     # 问答学习工作流（必读！）
├── frontend/                    # 前端技术栈分类
│   ├── [category]/
│   │   ├── questions.md         # 问答记录（问题+正确答案）
│   │   └── wrong_answers.md     # 错题本（需要复习的内容）
└── learning_progress.md         # 学习进度追踪
```

### 如何开始问答

1. **阅读工作流**：先查看 `.agent/workflows/quiz.md` 了解完整的问答流程
2. **选择分类**：用户会指定学习哪个技术栈
3. **检查错题本**：优先从 `wrong_answers.md` 中抽取问题进行复习
4. **提问并记录**：根据用户回答，更新 `questions.md` 或 `wrong_answers.md`

### 记录格式规范

#### questions.md 格式

```markdown
## Q: [问题内容]

**难度**：⭐ / ⭐⭐ / ⭐⭐⭐
**日期**：YYYY-MM-DD
**回答**：[用户的回答]
**评价**：✅ 正确 / ⚠️ 部分正确
**知识点补充**：[如有需要，补充相关知识]

---
```

#### wrong_answers.md 格式

```markdown
## Q: [问题内容]

**难度**：⭐ / ⭐⭐ / ⭐⭐⭐
**首次错误日期**：YYYY-MM-DD
**错误次数**：N
**用户错误回答**：[用户的错误回答]
**正确答案**：[详细的正确答案]
**关键知识点**：[需要掌握的核心概念]
**复习状态**：❌ 待复习 / 🔄 复习中 / ✅ 已掌握

---
```

### 复习策略

1. **新问题 vs 复习比例**：每 5 个新问题，穿插 1-2 个错题复习
2. **错题降级规则**：连续回答正确 2 次，错题状态可改为"已掌握"
3. **优先级**：错误次数多的题目优先复习

## 📊 当前支持的技术栈

| 分类       | 目录                   | 说明                        |
| ---------- | ---------------------- | --------------------------- |
| JavaScript | `frontend/javascript/` | ES6+、原型链、异步编程等    |
| CSS        | `frontend/css/`        | 布局、动画、响应式设计等    |
| HTML       | `frontend/html/`       | 语义化、表单、可访问性等    |
| React      | `frontend/react/`      | Hooks、状态管理、性能优化等 |
| Vue        | `frontend/vue/`        | 组合式 API、响应式原理等    |
| TypeScript | `frontend/typescript/` | 类型系统、泛型、工具类型等  |
